## 12. QA 风险点

| 风险 | 可能原因 | 验证方式 | 调整手段 |
|---|---|---|---|
| 四行流派碾压其他流派 | 四行伤害、I 生成强化过高 | 统计通关构筑分布 | 降低四行伤害或长条呼叫频率 |
| 连击流派过难 | 早期棋盘压力太高 | 观察 Combo 4+ 达成率 | 降低前 7 阶段垃圾行 |
| 高速阶段不可控 | 重力曲线上升过快 | 普通玩家第 15 阶段死亡率 | 降低 8-10 级速度或增加锁定延迟 |
| 锁定块导致死局 | 净化来源不足 | 记录锁定块同屏数量 | 限制生成上限，增加净化商店 |
| 主动技能必选 | 救场技能效率太高 | 统计技能购买率 | 提高能量消耗或冷却 |
| 经济膨胀 | 碎片产出叠乘过高 | 单局购买次数均值 | 调整碎片强化为加法或设上限 |
| 新手看不懂肉鸽选择 | 强化描述复杂 | 首局选择停留时间 | 简化前 3 阶段强化池 |
| Boss 过硬 | HP 与玩家伤害不匹配 | Boss 平均耗时 | 调整 HP 或压力间隔 |

## 13. 首轮调参目标

v0.2 调整记录：
- Stage 1 目标消行从 6 行下调至 4 行，目标是在首局前 30-60 秒内触发第一次强化选择，提升肉鸽反馈密度。

| 指标 | 目标值 |
|---|---:|
| 新手首次运行平均时长 | 6-10 分钟 |
| 熟练玩家普通难度通关率 | 25%-40% |
| 单局完整通关时长 | 20-25 分钟 |
| 每局获得强化数量 | 14-18 |
| 每局商店购买次数 | 4-7 |
| 第 1 章死亡率 | < 8% |
| 第 3 章前流派成型率 | > 70% |
| 最终 Boss 死亡占比 | 20%-35% |
